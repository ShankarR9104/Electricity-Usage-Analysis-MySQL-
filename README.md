# Electricity-Usage-Analysis-MySQL-
## Overview
This project is a MySQL-based electricity usage analytics system built using a star schema data warehouse design. A central fact table stores measurable consumption and billing metrics, while dimension tables (household, time, appliance, and environmental data) provide structured context for efficient querying and analysis.
The project demonstrates practical SQL skills including joins, aggregations, stored procedures, triggers, and relational design to analyze consumption patterns, automate billing calculations, and derive useful usage insights.
## Tools & Technologies
- MySQL
- SQL Queries (Joins, Aggregations, Subqueries)
- Data Modeling
## Dataset Used
Dataset - https://drive.google.com/drive/folders/19nWVjvza2tzJYDtOzyxQ4ligQMXvoon2?usp=sharing

  - Monthly usage
  - Meter readings
  - Time-based consumption
## Key Highlights

- ER design with foreign keys
- Advanced SQL queries
- Stored procedures & triggers
- Data analysis & reporting concepts

## Conclusion
This project successfully demonstrates the use of MySQL for analyzing real-world household electricity consumption data. By integrating multiple datasets into a well-structured relational database, it was possible to extract meaningful insights regarding usage patterns, monthly trends, and household-level consumption behavior. The analysis highlights how SQL can be effectively used not just for data storage, but also for data exploration and decision-making. The findings from this project can help in understanding energy usage habits, identifying peak consumption periods, and supporting efficient resource planning.
Overall, this project strengthens practical skills in data integration, database design, and advanced SQL querying, making it a strong foundation for data analytics and data science applications.
